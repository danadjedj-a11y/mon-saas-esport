# Création d’un utilisateur :
- Dans la création d’un compte, mettre le mail, mot de passe, date de naissance et pseudonyme.
- Ajouter également l’option pour mettre une photo de profil plus rapidement dès la création du compte.
- N’afficher que le pseudonyme par la suite sur toutes les pages.
- Mettre les profils en “public”, c’est-à-dire la possibilité de customiser son profil comme une page Steam (bio, statistiques, équipes etc.)
  
# Nom d’utilisateur RIOT GAMES :

- Pour rejoindre un tournoi VALORANT par exemple, il est impossible de rejoindre si le joueur n’a pas inscrit son nom d’utilisateur RIOT GAMES sur le site.
- Durant le lobby, le nom d’utilisateur RIOT GAMES sera inscrit en gris sombre et italique avec le logo RIOT GAMES en dessous du Pseudo du site.
- Valable également pour les autres tournois : Épic Games pour Rocket League, Ubisoft pour R6 etc.
  
# Invitation dans une équipe :

- Pour inviter un joueur dans une équipe, il vaudrait mieux que depuis la section “mon équipe”, il y ait un menu où tu puisses rechercher le joueur à inviter et que ce joueur reçoive une alerte d’invitation depuis le site et une notif par mail.
  
# Photo de profil :

 - Faire en sorte que tu dois “upload” la photo de profil plutôt que de mettre 2/3 un lien
   
# Page publique :

- Créer une page publique pour chaque équipe et chaque joueur
- Créer une page publique pour chaque match (à la manière de
TOORNAMENT) avec le pseudo des équipes face à face, la possibilité de
mettre à jour le score en temps réels par un admin pour les matchs
importants (finales), le stream en direct/le replay youtube, une section
“temps forts” avec automatiquement la publication des Tweets avec le #
du match et les clips en lien, et les réactions sur les réseaux avec le # du
match et/ou du tournoi (à mettre aussi sur la page principale ?) en temps
réel

# Gestion d’une équipe 

- Le Capitaine devrait pouvoir mettre des “grades” dans les équipes. Par
exemple : Manager (tu peux inviter/dégager des mecs), Coach (pareil que
manager)
